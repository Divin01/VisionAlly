# VisionAlly

VisionAlly is a mobile-first, multimodal AI assistant designed to provide real-time employment support and assistive coaching for individuals with disabilities.



The name is a strategic fusion of two core pillars:



Vision: Represents the technical use of LLM live video capabilities to "see" and analyze non-verbal cues like posture, eye contact, and confidence during interview preparation.



Ally: Reflects the tool's mission to act as a supportive, assistive system that enhances independence and inclusion for job seekers.
